Requirements:
Needs an installation of Web3js, testrpc, Metemask, Google Chrome

Running the application:
	- Open the terminal and run the following commands:
		- testrpc
		- cd DirectoryPathWhereFolderIsSaved/music
		- npm init
		- npm install lite-server --save-dev
		- npm install ethereum/web3.js --save
		- npm run dev

	- Login into your testrpc account using the Metamask plugin
	- Open index.html to buy a song
	- Open main.html to register a user
	- Open publish.html to publish a song

Built Using:
Solidity, Metmask, Web3js, HTML, CSS, Javascript
